About This Project

File Encryption & Decryption Tool System is a professional cybersecurity application designed for secure file handling and data protection. Built with modern technologies and OOP principles, it provides enterprise-level security features.

Technology Stack:

  • Frontend: HTML5, CSS3, JavaScript (Vanilla)
  • Backend: C++ (Encryption library & CLI/server)
  • Encryption: C++ with Caesar & XOR Cipher Algorithms
  • Database: SQLite with secure schema
  • Authentication: JWT-based token system
  • Design: Cybersecurity dark theme with Neon Blue accents
